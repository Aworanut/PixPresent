COMPONENT 04

Form Controls & Validation States (อินพุตฟอร์มและสถานะเส้นขอบบาง 1px)

ระบบอินพุตระดับสูงจะไม่ใช้บล็อกกล่องทึบหนา แต่จะอาศัยความพริ้วไหวของเส้นใต้ (Bottom hairline line) ขอบทองแชมเปญ และสัญญาณไฟสถานะสีเขียวมรกตหรือแดงชาดที่ละเอียดอ่อน:

01 // FOCUS / ACTIVE STATE

เส้นล่างจะเรืองแสงสีทองแชมเปญ (`#D4AF37`) เพื่อดึงโฟกัส

02 // VALIDATION SUCCESS STATE

สเตตัสผ่านจะแสดงเส้นสีเขียวมรกตบาง พร้อมจุดสัญญาณไฟในแนวระดับ

03 // ERROR STATE

เส้นใต้เปลี่ยนเป็นสีแดง Crimson อ่อนพริ้วไหว เพื่อการตอบรับที่ชัดเจนและสง่างาม
